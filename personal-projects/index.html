SozAI - AI Image Library

Overview: SozAI is a public web application providing high-quality, free-to-use AI-generated images for commercial and personal projects.

Role: Sole Developer (Design, Front-End, Infrastructure Setup, and Deployment)

Front-End Development

  • Core Technologies: HTML5, CSS3, and Vanilla JavaScript.
  • Dynamic Features: Implemented a client-side search engine that fetches and filters category data from static JSON files for instant, asynchronous results without server calls.

Cloud Infrastructure & Deployment

  • Hosting: Fully deployed and managed on AWS, demonstrating cloud deployment proficiency.
  • Storage & Delivery: Utilized Amazon S3 buckets for reliable static file storage, distributed globally via Amazon CloudFront for low-latency content delivery.
  • DNS & Security: Configured domain routing using Route 53 and managed SSL certificates via AWS Certificate Manager (ACM) to ensure HTTPS security.
